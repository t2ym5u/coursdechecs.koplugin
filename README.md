## Rules

Work through guided chess exercises. Each lesson presents a board position and an objective — make the correct moves to advance. Wrong moves are rejected. Progress through lessons to master chess fundamentals.

## Features

- **Structured lessons** — piece movements, tactics, checkmates
- **Step-by-step guidance** — instructions for each exercise
- **Progress tracking** — completed lessons are remembered
- **Auto-save** — current lesson position saved on exit

## Installation

1. Download `coursdechecs.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Chess Lessons**.

## Controls

| Action | How |
|--------|-----|
| Select a piece | Tap it |
| Make a move | Tap the destination square |
| Show rules | Tap **Rules** |
